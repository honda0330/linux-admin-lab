# Notes Linux Administration Lab

## File Management

Perintah yang dipelajari:

* ls
* cd
* pwd
* cp
* mv
* rm
* find
* grep

## User Management

Membuat user:

sudo adduser devops

Menambahkan ke grup sudo:

sudo usermod -aG sudo devops

## Bash Script

Script backup dibuat menggunakan:

tar -czf backup-$DATE.tar.gz project/

## Hasil

Backup berhasil dibuat dan repository berhasil di-push ke GitHub.
